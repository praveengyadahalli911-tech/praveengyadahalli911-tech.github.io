# Praveen G Y Portfolio

Full Stack Developer | AI Automation Engineer | Digital Solution Builder

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Firebase

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

```bash
vercel --prod
```

## Environment Variables

Copy `.env.example` to `.env` and fill in Firebase credentials.
